# Polish UI strings

Polish phrases used in Wisteria's interface. Personalization should stay lightweight — a few warm touches, not a bilingual manual.

## Convention

Every Polish string in this directory must include its **English translation in parentheses** on the same line or immediately below, so reviewers and future maintainers can read them without fluency in Polish.

Example format:

```
Czytam dla Ciebie (I read for you)
```

## Usage

- Boot messages, idle screens, and small UI labels only
- Reading content (book text) is not stored here
- Keep the total phrase count small; reading is the focus

## Phrases

Phrases will be added here as the UI is designed. None are finalized yet.
